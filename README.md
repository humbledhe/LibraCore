# LibraCore

A modern full-stack Library Management System built with FastAPI and React.

> 🚧 This project is currently under active development.

## Current Features

- User registration
- User authentication (Login)
- FastAPI REST API
- React + Vite frontend

## Planned Features

- Book management
- Author management
- Publisher management
- Category management
- Staff management
- Borrowing and returning books
- Search functionality
- Dashboard
- Reports

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic
- Pydantic

### Frontend
- React
- Vite
- React Router
- Axios
